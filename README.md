# 😭 Дипломный проект

👩🏻‍💻Проект с автоматизированными тестами сайта "Медси"

💉Медицинский холдинг МЕДСИ

(https://perm.medsi.ru/)

## ⓵ Первый этап 
В корневой папке проекта создать текстовый файл requirements.txt и прописать в нем все библиотеки, 
которые понадобятся при работе с автотестами. 

Далее - 💻 устанавливаем прописанные библиотеки на нашу рабочую машину, через команду в терминале:

pip install -r requirements.txt

## ⓶ Запуск автотестов 🤓

Для запуска в терминале прописываем следующие команды:
1) pytest --alluredir=./allure-results - запуск тестов, с указанием директории для сохранения отчётов

2) allure serve ./allure-results - просмотр Allure - отчета в браузере
